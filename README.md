# Crystall Apple Doll Shop (ca-shop)
BJD 娃娃配件的電商平台，具有 RWD 且使用 SPA，擁有基本的商品列表、購物車等功能。

### 開發特色
* 使用 Vue.js 開發
* 使用 Vuex 做狀態統一管理
* 使用 Vue-router 製作 SPA 頁面
* 使用 Sass 建構響應式版型 

### 前端功能
* 主視覺輪播區
* 商品列表頁
* 商品細節頁
* 購物車

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
